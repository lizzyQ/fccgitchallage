Timestamp Microservice

Hello there! Go ahead test your string to see if it's match a timestamp.

User stories:

Pass a string as part of the url, this program will check to see whether that string contains either a unix timestamp or a natural language date (example: January 1, 2016)
If it does, it returns both the Unix timestamp and the natural language form of that date.
If not, it returns null for those properties.

Example usage:

https://pickle-staircase.glitch.me/December%2015,%202015
https://pickle-staircase.glitch.me/1450137600

Example output:

{ "unix": 1450137600, "natural": "December 15, 2015" }
